# Project Byteboard – Custom Keyboard Build 🧠⌨️

## 🛠 What is this?
Project Byteboard is a custom 65% mechanical keyboard I’m building from scratch. Inspired by the HelloWord keyboard, this project enhances usability for programmers, creatives, and keyboard enthusiasts.

## 🎯 Goal
To create a custom, open-source keyboard with:
- 🌀 Rotary encoder knob
- 🖼 OLED screen for feedback
- ⌨️ Compact, hot-swappable layout
- 🧩 QMK/VIA compatibility
- 🛠 3D printed case with optional CNC design

## 💡 Why This Project?
This is more than a keyboard—it's a learning journey across:
- PCB design (KiCad)
- 3D modeling (CAD)
- Firmware development (QMK)
- Soldering and electronics
